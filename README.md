# Rock Paper Scissors in Haskell

[![Build Status](https://travis-ci.org/johanhaleby/rps-haskell.svg?branch=master)](https://travis-ci.org/johanhaleby/rps-haskell)

An attempt at Rock Paper Scissors in Haskell.

Initialize:

    $ stack setup
    $ stack build
    
Running: 
    
    $ stack run

Tests:

    $ stack test
